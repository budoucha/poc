ブラウザ間で WebRTC (Web Real-Time Communication) P2P 接続を確立し、ボタンを押すと相手の画面に通知を表示します。
URL の # フラグメントで SDP (Session Description Protocol) を受け渡す方式で、サーバは経由しません (STUN を除く)。

招待 URL を作って相手に送る側になります。相手が URL を開くと自動的にゲストとして参加します。

① 招待 URL を送る host

下の QR をカメラアプリで読み取るか、URL をコピーして相手に送ってください。

② 相手の返信 URL を貼り付ける host

相手から返ってきた返信内容をそのまま貼り付けてください。

返信 QR をカメラに映してください。
